# 🚀 Zomato Food Delivery Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing a Zomato Food Delivery dataset containing **123,657 rows and 12 columns** to extract meaningful insights and support data-driven decision making.

---

## 🔹 Project Workflow

- Imported dataset from **MySQL into Python (VS Code)** using MySQL Connector  
- Retrieved and analyzed data in **Jupyter Notebook**  
- Performed **data cleaning** using Pandas:
  - Handled missing values  
  - Corrected data types  
  - Standardized data  

- Stored cleaned dataset back into **MySQL using SQLAlchemy**  

---

## 🔍 Exploratory Data Analysis (EDA)

Performed detailed analysis including:

- **Univariate Analysis**
- **Bivariate Analysis**
- **Multivariate Analysis**

Used visualizations to identify patterns and trends in:
- Ratings  
- Votes  
- Cuisines  
- City distribution  

---

## 🗄️ SQL Analysis

Performed advanced SQL operations:

- GROUP BY  
- ORDER BY  
- JOIN  
- Subqueries  
- Window Functions  

---

## 📊 Power BI Dashboard

- Connected SQL dataset to Power BI using **ODBC Driver**  
- Built an **interactive dashboard (2 pages)**  
- Included:
  - KPIs  
  - Charts & Graphs  
  - Filters  
  - Bookmarks  
  - Navigation buttons & menu links
